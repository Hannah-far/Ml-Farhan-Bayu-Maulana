
Bike Sharing Analysis Dashboard

Deskripsi Proyek
Proyek ini menganalisis dataset Bike Sharing untuk memahami tren penyewaan sepeda berdasarkan faktor musim dan cuaca. Dilengkapi dengan implementasi Machine Learning sederhana (Linear Regression) untuk prediksi jumlah penyewaan.

Dataset
Dataset yang digunakan adalah Bike Sharing Dataset (day.csv).

Setup Environment

1. Install library yang dibutuhkan dengan perintah
   pip install -r requirements.txt

Run Dashboard
streamlit run dashboard.py

Struktur File

- dashboard.py: File utama aplikasi Streamlit.
- notebook.ipynb: Notebook berisi analisis data (EDA), visualisasi, dan pemodelan.
- day.csv: Dataset yang digunakan.
- requirements.txt: Daftar library yang harus diinstall.
